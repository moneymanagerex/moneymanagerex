Établir un budget avec MMEX

Options d'établissement de budget

L'option View → Budgets: en années fiscales peut aussi être utilisé pour passé d'une vue à l'autre.

MMEX permet d'établir un budget annualisé et/ou mensualisé. Ceci peut vous permettre de confronter votre budget à votre consommation réelle.

Les budgets peuvent être affichés en années civiles ou en années fiscales. Utilisez la boîte de dialogue pour paramétrer le début d'une année financière, ainsi que les paramètres permanents.

Créer un Nouveau Compte

Pour établir un budget, faites un clique droit sur le nœud 'Établir un budget' et ajoutez un budget mensualisé ou annualisé.

Éditer les catégories de budget

Dès qu'une année a été ajoutée, sélectionnez l'année souhaitée pour afficher la fenêtre d'édition du budget.

Chaque catégorie peut être éditée en double-cliquant sur celles-ci. Cela doit être fait pour toutes les catégories.

Panneau de configuration du budget

Cela devient le budget pour cette année là. Les mois et/ou les années qui suivent peuvent alors en découler.

Catégorie Sous-Totaux Totaux sont affichés pour chaque catégorie principale.

Lors de la configuration d'un budget, les Totaux peuvent êtres déactivés en allant dans le menu: Vues → Configuration des Budgets: Sans totaux.

Rapports des budgets

Il est possible de confronter l'argent dépensé par rapport au budget prévionnel en utilisant les rapports sous 'Budget'.
