Presupuestos

Inicio
MMEX permite la configuración de un presupuesto para un determinado año y/o mes.
Esto permite comparar lo que gasta con su presupuesto.

Los presupuestos se pueden mostrar en años de calendario o años fiscales.

Utilice las Opciones para seleccionar el inicio del año fiscal, y cualquier cambio permanente que necesite.
La opción Ver->Presupuestos: Como años fisales también se puede usar para cambiar entre las 2 vistas.
Crear nuevos presupuestos:

Para configurar un presupuesto haga click derecho en la rama 'Presupuestos' del árbol y añada un mes o un año.



Edición de las categorías del presupuesto:

Una vez añadido el año, selecciónelo para mostrar la configuración del presupuesto.
Se puede editar cada categoría haciendo doble click sobre ella.



Esto se realiza para todas las subsiguientes categorías.

Configuración del presupuesto:

Esto se convierte en el presupuesto para ese año. Los meses y/o años posteriores pueden derivar de este año.



El resumen total de categoría se muestra por cada categoría principial.
Al configurar un presupuesto, el resumen total del presupuesto se puede desactivar mediante la opción del menú:
Ver->Organización de presupuesto: Sin resúmenes


Mediante los informes bajo 'Presupuestos', ahora es posible comparar cómo ha gastado el dinero frente a su presupuesto real.
